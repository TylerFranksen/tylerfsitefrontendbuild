This Serverless Resume Site

2022

My website is meant to be a constant work in progress, so please keep in mind this summary is just for the resume portion of the site. The contents of the webpage are all hosted in an S3 bucket with automated CI/CD deployments for updating/religating versions of the vital file structure within the bucket. This website utilizes Lambda/DynamoDB/JavaScript to count the number of visitors the site has received. In order to achieve this, a Lambda function first requests the contents of a data table hosted in DynamoDB and checks the value for an internal parameter to see the current visit count. The function then adds 1 to the value of the contained parameter and sends it back to the DynamoDB table, JavaScript is used to find the value within HTML that we are inserting the DynamoDB Visitor Count contents into. I have also connected QuickSight to play around with the dashboard in order to gain familiarity later in a professional role.
