FBI ETL Pipeline

2022

This project was multiple passions conglomerated, I wanted to develop a serverless and automated method for accurately measuring how many people in the US have a firearm and what kind potentially. I originally wanted to see how many people were possibly owning a firearm in the United States, and that eventually brought me to researching the background check process and how intertwined it is with owning a firearm. The pipeline scrapes data from the NCIS report which gets updated every few weeks and updates a DynamoDB table with proper attributes for tracking citizens and their respective firearms purchases. This does not tell me how many guns one citizen owns but rather, total owned for each category of firearm/transaction in each state for each month of the year.

This Serverless Resume Site

2022

My website is meant to be a constant work in progress, so please keep in mind this summary is just for the resume portion of the site. The contents of the webpage are all hosted in an S3 bucket with automated CI/CD deployments for updating/religating versions of the vital file structure within the bucket. This website utilizes Lambda/DynamoDB/JavaScript to count the number of visitors the site has received. In order to achieve this, a Lambda function first requests the contents of a data table hosted in DynamoDB and checks the value for an internal parameter to see the current visit count. The function then adds 1 to the value of the contained parameter and sends it back to the DynamoDB table, JavaScript is used to find the value within HTML that we are inserting the DynamoDB Visitor Count contents into.
